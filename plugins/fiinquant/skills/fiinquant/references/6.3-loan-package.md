# 7.3.1. Gói vay thường

**Tên hàm:** `client_order.get_loan_packages()`

**Mục đích:**
Lấy danh sách các gói vay (margin hoặc thường) áp dụng cho tài khoản chứng khoán cơ sở, bao gồm thông tin về tỷ lệ vay, lãi suất, phí giao dịch và danh sách mã được hỗ trợ.

**Tham số:**

| Tham số   | Kiểu dữ liệu | Bắt buộc | Mô tả |
|----------|-------------|----------|------|
| account_id | str | Có | Mã tiểu khoản cần lấy thông tin gói vay |

**Kết quả trả về:**
- Trả về danh sách các object `BaseLoanPackage`
- Mỗi object biểu diễn một gói vay

**Thuộc tính dữ liệu:**

| Tên thuộc tính | Kiểu dữ liệu | Mô tả |
|---------------|-------------|------|
| id | str | Mã định danh gói vay |
| type | str | Loại gói vay: M = Margin, N = Normal |
| initial_rate | float | Tỷ lệ vay ban đầu |
| maintenance_rate | float | Tỷ lệ duy trì |
| interest_rate | float | Lãi suất vay (%/năm) |
| preferential_interest_rate | float | Lãi suất ưu đãi |
| allow_extend | bool | Có cho phép gia hạn hay không |
| allow_early_payment | bool | Có cho phép tất toán trước hạn hay không |
| buy_fee | float | Phí giao dịch mua |
| sell_fee | float | Phí giao dịch bán |
| tickers | list hoặc str | Danh sách mã áp dụng |
| error | str hoặc None | Thông tin lỗi |
| status_code | int | Mã trạng thái |
| num_tickers | int | Số lượng mã áp dụng |

**Ví dụ sử dụng:**

```python
username_dnse = 'REPLACE_WITH_YOUR_DNSE_USERNAME'
password_dnse = 'REPLACE_WITH_YOUR_DNSE_PASSWORD'

client_order = client.FiinQuantConnector(
    broker='DNSE',
    username=username_dnse,
    password=password_dnse,
    smart_otp=True
).login()

loan_packages = client_order.get_loan_packages(account_id='0001009212')

for i in loan_packages:
    print(i.summary())
```
**Output mẫu:**
```json
{
    "id": 7905,
    "type": "M",
    "initial_rate": 0.5,
    "maintenance_rate": 0.4,
    "interest_rate": 0.12,
    "preferential_interest_rate": 0.12,
    "allow_extend": true,
    "allow_early_payment": true,
    "buy_fee": 0.0007,
    "sell_fee": 0.0007,
    "tickers": ["ACB", "BCC", "BID", "CTG", "FPT", "HPG"],
    "error": null,
    "status_code": 200,
    "num_tickers": 19
}
```

# 7.3.2. Gói vay phái sinh

**Tên hàm:** `client_order.get_derivative_loan_packages()`

**Mục đích:**
Lấy danh sách các gói ký quỹ áp dụng cho tài khoản phái sinh, bao gồm tỷ lệ ký quỹ, mức xử lý (liquidation), phí giao dịch và danh sách loại hợp đồng được hỗ trợ.

**Tham số:**

| Tham số   | Kiểu dữ liệu | Bắt buộc | Mô tả |
|----------|-------------|----------|------|
| account_id | str | Có | Mã tiểu khoản phái sinh |

**Kết quả trả về:**
- Trả về danh sách các object `BaseDerivativeLoanPackage`

**Thuộc tính dữ liệu:**

| Tên thuộc tính | Kiểu dữ liệu | Mô tả |
|---------------|-------------|------|
| id | str | Mã định danh gói vay |
| initial_rate | float | Tỷ lệ ký quỹ ban đầu |
| maintenance_rate | float | Tỷ lệ duy trì |
| liquid_rate | float | Tỷ lệ xử lý bắt buộc |
| trading_fee | float | Phí giao dịch |
| trading_fee_daily_close | float | Phí giao dịch đóng vị thế hàng ngày |
| symbol_types | list hoặc str | Danh sách loại hợp đồng |
| error | str hoặc None | Thông tin lỗi |
| num_symbol_types | int | Số lượng loại hợp đồng |
| status_code | int | Mã trạng thái |

**Ví dụ sử dụng:**

```python
username_dnse = 'REPLACE_WITH_YOUR_DNSE_USERNAME'
password_dnse = 'REPLACE_WITH_YOUR_DNSE_PASSWORD'

client_order = client.FiinQuantConnector(
    broker='DNSE',
    username=username_dnse,
    password=password_dnse,
    smart_otp=True
).login()

der_loans = client_order.get_derivative_loan_packages(account_id='0001009212')

for i in der_loans:
    print(i.summary())
```
**Output mẫu:**
```json
{
    "id": 2278,
    "initial_rate": 0.1848,
    "maintenance_rate": 0.1771,
    "liquid_rate": 0.1731,
    "trading_fee": 500.0,
    "trading_fee_daily_close": 500.0,
    "symbol_types": ["VN30F1M", "VN30F2M", "VN30F1Q"],
    "error": null,
    "num_symbol_types": 6,
    "status_code": 200
}
```
