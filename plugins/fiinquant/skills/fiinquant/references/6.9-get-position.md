# 7.9. Lấy vị thế

**Tên hàm:**
- Cơ sở: `orderbook.get_positions(...)`
- Phái sinh: `orderbook_der.get_positions(...)`

**Mục đích:**
Lấy danh sách các vị thế hiện có của tài khoản giao dịch, bao gồm vị thế đang mở hoặc đã đóng, cùng với thông tin lãi/lỗ thực hiện và chưa thực hiện.

**Tham số:**

| Tên tham số | Kiểu dữ liệu | Bắt buộc | Mô tả |
|---|---|---|---|
| ticker | str hoặc None | Không | Mã chứng khoán hoặc mã phái sinh cần lọc. Nếu không truyền, trả về tất cả mã |
| status | str hoặc None | Không | Trạng thái vị thế cần lọc. Có thể là `"OPEN"` hoặc `"CLOSED"` |
| side | str hoặc None | Không | Chiều giao dịch của vị thế. Có thể là `"buy"` hoặc `"sell"` |

**Kết quả trả về (Returns):**
- Trả về danh sách object vị thế
- Mỗi phần tử biểu diễn một vị thế giao dịch
- Có thể gọi `.summary()` trên từng object để hiển thị nhanh dữ liệu
- Dữ liệu bao gồm trạng thái vị thế, giá vốn, giá thị trường, lãi/lỗ thực hiện và chưa thực hiện

**Thuộc tính dữ liệu:**

| Tên thuộc tính | Kiểu dữ liệu | Mô tả |
|---|---|---|
| id | int | Mã định danh của vị thế |
| ticker | str | Mã chứng khoán hoặc mã phái sinh |
| account_no | str | Mã tiểu khoản giao dịch |
| side | str | Chiều giao dịch của vị thế |
| status | str | Trạng thái vị thế, ví dụ: `OPEN`, `CLOSED` |
| quantity | int | Khối lượng hiện tại của vị thế |
| cost_price | float | Giá vốn trung bình của vị thế |
| market_price | float | Giá thị trường hiện tại |
| accumulate_quantity | int | Tổng khối lượng tích lũy |
| break_even_price | float | Giá hòa vốn của vị thế |
| realized_profit | float | Lãi/lỗ đã thực hiện |
| unrealized_profit | float | Lãi/lỗ chưa thực hiện |
| created_date | str | Thời điểm tạo vị thế |
| error | str hoặc None | Thông tin lỗi nếu có |
| status_code | int | Mã code, nếu thành công là 200 |

**Ví dụ cho cơ sở:**

```python
username_dnse = 'REPLACE_WITH_YOUR_DNSE_USERNAME'
password_dnse = 'REPLACE_WITH_YOUR_DNSE_PASSWORD'

client_order = client.FiinQuantConnector(
    broker='DNSE',
    username=username_dnse,
    password=password_dnse,
    smart_otp=True
).login()

orderbook = client_order.get_orderbook(
    account_id='0001009212',
    loan_id='13720'
)

positions = orderbook.get_positions()
for i in positions:
    print(i.summary())
```

**Ví dụ cho phái sinh:**
```python
username_dnse = 'REPLACE_WITH_YOUR_DNSE_USERNAME'
password_dnse = 'REPLACE_WITH_YOUR_DNSE_PASSWORD'

client_order = client.FiinQuantConnector(
    broker='DNSE',
    username=username_dnse,
    password=password_dnse,
    smart_otp=True
).login()

orderbook_der = client_order.get_derivative_orderbook(
    account_id='0001009212',
    loan_id='2278'
)

for i in derivative_positions:
    print(i.summary())
```
