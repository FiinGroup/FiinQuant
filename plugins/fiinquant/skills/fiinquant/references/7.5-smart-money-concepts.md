# 8.5.1. Fair Value Gap (FVG)

**Tên hàm:**
- `fi.fvg(open, high, low, close, join_consecutive=True)`
- `fi.fvg_top(open, high, low, close, join_consecutive=True)`
- `fi.fvg_bottom(open, high, low, close, join_consecutive=True)`
- `fi.fvg_mitigatedIndex(open, high, low, close, join_consecutive=True)`

**Mục đích:**
Xác định vùng giá chưa được lấp đầy trên biểu đồ do mất cân bằng giữa cung và cầu, còn gọi là Fair Value Gap.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| open | Cột chứa giá mở cửa | pandas.Series |  |
| high | Cột chứa giá cao nhất | pandas.Series |  |
| low | Cột chứa giá thấp nhất | pandas.Series |  |
| close | Cột chứa giá đóng cửa | pandas.Series |  |
| join_consecutive | Nếu có nhiều khoảng FVG liên tiếp, chúng sẽ được gộp lại thành một | bool | True |

**Kết quả trả về (Returns):**
- `fvg()`: trả về `pandas.Series` xác định vùng FVG
- `fvg_top()`: trả về `pandas.Series` biên trên của FVG
- `fvg_bottom()`: trả về `pandas.Series` biên dưới của FVG
- `fvg_mitigatedIndex()`: trả về `pandas.Series` chỉ số nến mà FVG được lấp

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['fvg'] = fi.fvg(df['open'], df['high'], df['low'], df['close'], join_consecutive=True)
print(df)

```
# 8.5.2. Swing Highs and Lows

**Tên hàm:**
- `fi.swing_HL(open, high, low, close, swing_length=50)`
- `fi.swing_level(open, high, low, close, swing_length=50)`

**Mục đích:**
Xác định các đỉnh xoay và đáy xoay trên biểu đồ để phục vụ phân tích cấu trúc thị trường và các tín hiệu đảo chiều.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| open | Cột chứa giá mở cửa | pandas.Series |  |
| high | Cột chứa giá cao nhất | pandas.Series |  |
| low | Cột chứa giá thấp nhất | pandas.Series |  |
| close | Cột chứa giá đóng cửa | pandas.Series |  |
| swing_length | Số lượng nến cần xét về trước và sau để xác định đỉnh hoặc đáy xoay | int | 50 |

**Kết quả trả về (Returns):**
- `swing_HL()`: trả về `pandas.Series` xác định swing high hoặc swing low
- `swing_level()`: trả về `pandas.Series` mức giá swing tương ứng

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['swing_HL'] = fi.swing_HL(df['open'], df['high'], df['low'], df['close'], swing_length=50)
print(df)

```
# 8.5.3. Break of Structure (BOS) & Change of Character (CHoCH)

**Tên hàm:**
- `fi.break_of_structure(open, high, low, close, close_break=True, swing_length=50)`
- `fi.chage_of_charactor(open, high, low, close, close_break=True, swing_length=50)`
- `fi.bos_choch_level(open, high, low, close, close_break=True, swing_length=50)`
- `fi.broken_index(open, high, low, close, close_break=True, swing_length=50)`

**Mục đích:**
Xác định các điểm phá vỡ cấu trúc thị trường (BOS) và thay đổi đặc tính xu hướng (CHoCH) để nhận diện tiếp diễn hoặc đảo chiều xu hướng.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Mặc định |
|---|---|---|---|
| open | Cột chứa giá mở cửa | pandas.Series |  |
| high | Cột chứa giá cao nhất | pandas.Series |  |
| low | Cột chứa giá thấp nhất | pandas.Series |  |
| close | Cột chứa giá đóng cửa | pandas.Series |  |
| close_break | Nếu True, xác nhận dựa trên giá đóng cửa. Nếu False, xác nhận dựa trên mức cao hoặc thấp của nến | bool | True |
| swing_length | Số lượng nến cần xét về trước và sau để xác định đỉnh hoặc đáy xoay | int | 50 |

**Kết quả trả về (Returns):**
- `break_of_structure()`: trả về `pandas.Series` tín hiệu BOS
- `chage_of_charactor()`: trả về `pandas.Series` tín hiệu CHoCH
- `bos_choch_level()`: trả về `pandas.Series` mức giá BOS hoặc CHoCH
- `broken_index()`: trả về `pandas.Series` vị trí nến phá vỡ

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['break_of_structure'] = fi.break_of_structure(df['open'], df['high'], df['low'], df['close'], swing_length=50)
df['chage_of_charactor'] = fi.chage_of_charactor(df['open'], df['high'], df['low'], df['close'])
print(df)

```
# 8.5.4. Order Blocks

**Tên hàm:**
- `fi.ob(open, high, low, close, volume, close_mitigation=False, swing_length=50)`
- `fi.ob_top(open, high, low, close, volume, close_mitigation=False, swing_length=50)`
- `fi.ob_bottom(open, high, low, close, volume, close_mitigation=False, swing_length=50)`
- `fi.ob_volume(open, high, low, close, volume, close_mitigation=False, swing_length=50)`
- `fi.ob_mitigated_index(open, high, low, close, volume, close_mitigation=False, swing_length=50)`
- `fi.ob_percetage(open, high, low, close, volume, close_mitigation=False, swing_length=50)`

**Mục đích:**
Xác định vùng giá mà các tổ chức lớn đã giao dịch mạnh, từ đó tìm các vùng Order Block tiềm năng để theo dõi phản ứng giá.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Mặc định |
|---|---|---|---|
| open | Cột chứa giá mở cửa | pandas.Series |  |
| high | Cột chứa giá cao nhất | pandas.Series |  |
| low | Cột chứa giá thấp nhất | pandas.Series |  |
| close | Cột chứa giá đóng cửa | pandas.Series |  |
| volume | Cột chứa khối lượng giao dịch | pandas.Series |  |
| close_mitigation | Nếu True, xác nhận mitigation dựa trên giá đóng cửa. Nếu False, xác nhận theo mức cao hoặc thấp | bool | False |
| swing_length | Số lượng nến cần xét | int | 50 |

**Kết quả trả về (Returns):**
- `ob()`: trả về `pandas.Series` xác định vùng Order Block
- `ob_top()`: trả về `pandas.Series` biên trên Order Block
- `ob_bottom()`: trả về `pandas.Series` biên dưới Order Block
- `ob_volume()`: trả về `pandas.Series` khối lượng liên quan đến Order Block
- `ob_mitigated_index()`: trả về `pandas.Series` vị trí mitigation
- `ob_percetage()`: trả về `pandas.Series` tỷ lệ liên quan đến Order Block

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['ob'] = fi.ob(df['open'], df['high'], df['low'], df['close'], df['volume'], close_mitigation=False, swing_length=40)
df['ob_volume'] = fi.ob_volume(df['open'], df['high'], df['low'], df['close'], df['volume'])
print(df)

```
# 8.5.5. Liquidity

**Tên hàm:**
- `fi.liquidity(open, high, low, close, range_percent=0.01, swing_length=50)`

- `fi.liquidity_swept(open, high, low, close, range_percent=0.01, swing_length=50)`

**Mục đích:**
Xác định các vùng thanh khoản, nơi có nhiều mức đỉnh hoặc đáy tập trung trong một phạm vi nhỏ, thường là khu vực tích lũy lệnh.

| Tên tham số | Mô tả | Kiểu dữ liệu | Mặc định |
|---|---|---|---|
| open | Cột chứa giá mở cửa | pandas.Series |  |
| high | Cột chứa giá cao nhất | pandas.Series |  |
| low | Cột chứa giá thấp nhất | pandas.Series |  |
| close | Cột chứa giá đóng cửa | pandas.Series |  |
| range_percent | Phần trăm phạm vi giá được sử dụng để xác định thanh khoản | float | 0.01 |
| swing_length | Số lượng nến cần xét về trước và sau để xác định đỉnh hoặc đáy xoay | int | 50 |

**Kết quả trả về (Returns):**
- `liquidity()`: trả về `pandas.Series` xác định vùng thanh khoản
- `liquidity_level()`: trả về `pandas.Series` mức giá thanh khoản
- `liquidity_end()`: trả về `pandas.Series` vị trí kết thúc vùng thanh khoản
- `liquidity_swept()`: trả về `pandas.Series` tín hiệu quét thanh khoản

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['liquidity'] = fi.liquidity(df['open'], df['high'], df['low'], df['close'])
print(df)
