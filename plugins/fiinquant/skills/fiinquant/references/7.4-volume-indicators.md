# 8.4.1. MFI (Money Flow Index)

**Tên hàm:** `fi.mfi(high, low, close, volume, window=14)`

**Mục đích:**
Tính chỉ số dòng tiền MFI để phản ánh sức mạnh dòng tiền của cổ phiếu trong một khoảng thời gian nhất định dựa trên giá và khối lượng giao dịch.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| high | Cột dữ liệu chứa các giá trị cột giá cao nhất để tính toán MFI | pandas.Series |  |
| low | Cột dữ liệu chứa các giá trị cột giá thấp nhất để tính toán MFI | pandas.Series |  |
| close | Cột dữ liệu chứa các giá trị cột giá đóng cửa để tính toán MFI | pandas.Series |  |
| volume | Cột dữ liệu chứa các giá trị cột khối lượng giao dịch để tính toán MFI | pandas.Series |  |
| window | Số lượng điểm dữ liệu sử dụng trong phép tính MFI | int | 14 |

**Kết quả trả về (Returns):**
- Trả về `pandas.Series`
- Mỗi giá trị là chỉ số MFI tương ứng tại từng thời điểm

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['mfi'] = fi.mfi(df['high'], df['low'], df['close'], df['volume'], window=14)
print(df)

```

# 8.4.4. Volume Profile

**Tên hàm:** `VolumeProfile(close, volume, high=None, low=None, bins=20, use_typical=False)`

**Mục đích:**
Phân tích phân bố khối lượng giao dịch tại các mức giá khác nhau trong một giai đoạn cụ thể để xác định các vùng giá được giao dịch nhiều hoặc ít.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| close | Cột giá đóng cửa dùng để xác định mức giá giao dịch | pandas.Series |  |
| volume | Cột khối lượng giao dịch tương ứng với từng mức giá | pandas.Series |  |
| high | Cột giá cao nhất, sử dụng khi dùng Typical Price | pandas.Series | None |
| low | Cột giá thấp nhất, sử dụng khi dùng Typical Price | pandas.Series | None |
| bins | Số lượng khoảng giá để chia biểu đồ Volume Profile | int | 20 |
| use_typical | Nếu `True`, dùng Typical Price = `(High + Low + Close)/3` thay cho Close | bool | False |

**Kết quả trả về (Returns):**
- Trả về `pandas.DataFrame`
- Mỗi dòng biểu diễn một khoảng giá và tổng khối lượng giao dịch tương ứng
- Có thể sắp xếp để xác định các vùng giá có thanh khoản lớn nhất

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
vp = VolumeProfile(
    close=data["close"],
    volume=data["volume"],
    high=data["high"],
    low=data["low"],
    bins=20,
    use_typical=True
)
print(vp.sort_values("volume", ascending=False))

```

| volume | Cột khối lượng giao dịch trong mỗi phiên | pandas.Series |  |

**Kết quả trả về (Returns):**
- Trả về `pandas.Series`
- Mỗi giá trị là A/D Line tích lũy tương ứng tại từng thời điểm

```

# 8.4.7. Point of Control (POC)

**Tên hàm:** `fi.poc(close, volume, high=None, low=None, bins=20, use_typical=False)`

**Mục đích:**
Xác định khoảng giá có khối lượng giao dịch lớn nhất trong Volume Profile, từ đó tìm ra vùng Point of Control (POC) quan trọng của thị trường.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| close | Cột giá đóng cửa dùng để xác định mức giá giao dịch | pandas.Series |  |
| volume | Cột khối lượng giao dịch tương ứng với từng mức giá | pandas.Series |  |
| high | Cột giá cao nhất, sử dụng khi dùng Typical Price | pandas.Series | None |
| low | Cột giá thấp nhất, sử dụng khi dùng Typical Price | pandas.Series | None |
| bins | Số lượng khoảng giá để chia biểu đồ Volume Profile | int | 20 |
| use_typical | Nếu `True`, dùng Typical Price = `(High + Low + Close)/3` thay cho Close | bool | False |

**Kết quả trả về (Returns):**
- Trả về `pandas.DataFrame`
- Gồm 3 cột:
  - `poc_lower_price`: cận dưới của khoảng giá có khối lượng giao dịch lớn nhất
  - `poc_upper_price`: cận trên của khoảng giá có khối lượng giao dịch lớn nhất
  - `poc_volume`: khối lượng giao dịch của khoảng giá đó

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
data[["poc_lower_price", "poc_upper_price", "poc_volume"]] = fi.poc(
    data["close"],
    data["volume"],
    data["high"],
    data["low"],
    bins=20,
    use_typical=True
)
print(data)
