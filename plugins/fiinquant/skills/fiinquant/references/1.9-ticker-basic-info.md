# Lấy dữ liệu thông tin cơ bản của cổ phiếu (tên DN, sàn GD, ngành)

**Tên hàm:** `client.BasicInfor(tickers=tickers).get()`

**Mục đích:**
Lấy dữ liệu thông tin cơ bản của một hoặc nhiều cổ phiếu, bao gồm tên doanh nghiệp, mã số thuế, sàn giao dịch và thông tin ngành theo các cấp ICB.

**Tham số (Parameters):**

| Tên tham số | Mô tả | Kiểu dữ liệu | Mặc định | Bắt buộc |
|---|---|---|---|---|
| tickers | Danh sách mã được viết in hoa | str hoặc list |  | Có |

**Kết quả trả về (Returns):**
- Kiểu dữ liệu: bảng dữ liệu / danh sách bản ghi
- Class nhận dữ liệu là `BasicInfor`
- Phương thức nhận dữ liệu là `get()`

### Dữ liệu có các thuộc tính

| Tên thuộc tính | Mô tả | Kiểu dữ liệu |
|---|---|---|
| ticker | Tên mã | str |
| taxCode | Mã số thuế doanh nghiệp | str |
| organizationName | Tên đầy đủ doanh nghiệp | str |
| organizationShortName | Tên viết tắt doanh nghiệp | str |
| exchangeCode | Tên sàn giao dịch | str |
| sector | Tên ngành | str |
| icbNameL5 | Tên ngành cấp 5 dùng để làm tham số | str |
| icbNameL4 | Tên ngành cấp 4 dùng để làm tham số | str |
| icbNameL3 | Tên ngành cấp 3 dùng để làm tham số | str |
| icbNameL2 | Tên ngành cấp 2 dùng để làm tham số | str |
| icbNameL1 | Tên ngành cấp 1 dùng để làm tham số | str |

**Ví dụ:**
```python
from FiinQuantX import FiinSession

username = 'REPLACE_WITH_YOUR_USER_NAME'
password = 'REPLACE_WITH_YOUR_PASS_WORD'

client = FiinSession(username=username, password=password).login()

# Lấy thông tin cơ bản của danh sách cổ phiếu VN30
tickers = client.TickerList(ticker="VN30")

data = client.BasicInfor(
    tickers=tickers
).get()

print(data)
