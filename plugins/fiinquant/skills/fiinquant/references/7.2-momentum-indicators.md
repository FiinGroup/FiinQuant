# 8.2.1. RSI (Relative Strength Index)

**Tên hàm:** `fi.rsi(column, window=14)`

**Mục đích:**
Đo lường tốc độ và biên độ biến động giá gần đây để xác định trạng thái quá mua hoặc quá bán của tài sản.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| column | Cột dữ liệu chứa các giá trị để tính toán RSI | pandas.Series |  |
| window | Số lượng điểm dữ liệu sử dụng trong phép tính RSI | int | 14 |

**Kết quả trả về (Returns):**
- Trả về `pandas.Series`
- Giá trị RSI dao động trong khoảng từ 0 đến 100

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['rsi'] = fi.rsi(df['close'], window=14)
print(df)

```
# 8.2.2. RSI Divergence (Relative Strength Index Divergence)

|---|---|---|---|
| column | Cột dữ liệu chứa các giá trị để tính toán RSI phân kỳ | pandas.Series |  |
| fast_window | Số lượng điểm dữ liệu sử dụng trong phép tính RSI nhanh | int | 5 |
| slow_window | Số lượng điểm dữ liệu sử dụng trong phép tính RSI chậm | int | 14 |

**Kết quả trả về (Returns):**
- Trả về `pandas.Series`
- Mỗi giá trị biểu diễn mức độ phân kỳ RSI theo thời gian

**Ví dụ sử dụng:**

```python

df['rsi_divergence'] = fi.rsi_divergence(df['close'], fast_window=5, slow_window=14)

```
# 8.2.3. Stochastic

**Tên hàm:**
- `fi.stoch(high, low, close, window=14)`
- `fi.stoch_signal(high, low, close, window=14, smooth_window=3)`

**Mục đích:**
Đo lường vị trí của giá đóng cửa hiện tại trong phạm vi giá cao nhất và thấp nhất của một giai đoạn để xác định động lượng và vùng quá mua/quá bán.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| high | Cột dữ liệu chứa các giá trị giá cao nhất | pandas.Series |  |
| low | Cột dữ liệu chứa các giá trị giá thấp nhất | pandas.Series |  |
| close | Cột dữ liệu chứa các giá trị giá đóng cửa | pandas.Series |  |
| window | Số lượng điểm dữ liệu sử dụng trong phép tính Stochastic | int | 14 |
| smooth_window | Số lượng điểm dữ liệu dùng để làm mượt đường tín hiệu | int | 3 |

**Kết quả trả về (Returns):**
- `stoch()`: trả về `pandas.Series` của đường %K
- `stoch_signal()`: trả về `pandas.Series` của đường %D

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['stoch'] = fi.stoch(df['high'], df['low'], df['close'], window=14)
df['stoch_signal'] = fi.stoch_signal(df['high'], df['low'], df['close'], window=14, smooth_window=3)
print(df)

```

# 8.2.5. PPO (Percentage Price Oscillator)

**Tên hàm:**
- `fi.ppo(close, window_slow=26, window_fast=12, window_sign=9)`
- `fi.ppo_hist(close, window_slow=26, window_fast=12, window_sign=9)`
- `fi.ppo_signal(close, window_slow=26, window_fast=12, window_sign=9)`

**Mục đích:**
Đo lường chênh lệch phần trăm giữa hai đường EMA để đánh giá động lượng giá và tín hiệu giao cắt.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| close | Cột giá đóng cửa của tài sản dùng để tính PPO | pandas.Series |  |
| window_slow | Số kỳ của EMA chậm | int | 26 |
| window_fast | Số kỳ của EMA nhanh | int | 12 |
| window_sign | Số kỳ dùng để tính đường tín hiệu | int | 9 |

**Kết quả trả về (Returns):**
- `ppo()`: trả về `pandas.Series` của PPO
- `ppo_hist()`: trả về `pandas.Series` của histogram PPO
- `ppo_signal()`: trả về `pandas.Series` của đường tín hiệu PPO

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['ppo'] = fi.ppo(df['close'], window_slow=26, window_fast=12, window_sign=9)
df['ppo_hist'] = fi.ppo_hist(df['close'], window_slow=26, window_fast=12, window_sign=9)
df['ppo_signal'] = fi.ppo_signal(df['close'], window_slow=26, window_fast=12, window_sign=9)
print(df)

```
