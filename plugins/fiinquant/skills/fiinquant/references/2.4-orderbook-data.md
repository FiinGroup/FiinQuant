# 2.4. Hàm dữ liệu sổ lệnh

Mô tả cách sử dụng thư viện sau khi người dùng đã đăng nhập. Các ví dụ chi tiết được nêu ra ở cuối chương này.

**Tên hàm:** `client.BidAsk(tickers=tickers, callback=callback)`

**Mục đích:**
Nhận dữ liệu sổ lệnh realtime cho một hoặc nhiều mã chứng khoán, chỉ số, phái sinh hoặc chứng quyền. Dữ liệu sẽ được xử lý thông qua hàm callback do người dùng tự định nghĩa.

**Tham số (Parameters):**

| Tên tham số | Mô tả | Kiểu dữ liệu | Bắt buộc |
|---|---|---|---|
| tickers | Tên mã, bao gồm mã chứng khoán, mã chỉ số, mã phái sinh và mã chứng quyền. Các mã này được viết in hoa. | str hoặc list | Có |
| callback | Là hàm do người dùng tự định nghĩa để thao tác với dữ liệu. | function | Có |

**Phương thức của class nhận dữ liệu:**

| Tên phương thức | Mô tả |
|---|---|
| start() | Bắt đầu kết nối và nhận dữ liệu. |
| stop() | Dừng kết nối. |

**Hàm callback**

Là phương thức do người dùng tự định nghĩa để thao tác với dữ liệu, sẽ được truyền vào khi khởi tạo đối tượng nhận dữ liệu như là một đối số. Phương thức này sẽ có dạng như sau:

```python
# pseudocode
def name_of_callback(data: BidAskData):
    # do something

event = client.BidAsk(
    tickers=['ticker_1', 'ticker_2', 'ticker_n'],
    callback=name_of_callback
)
event.start()

# tickers can include ticker, covered warrant, index and derivative.
event = client.BidAsk(
    tickers=['ticker_1', 'ticker_2', 'ticker_n'],
    callback=name_of_callback
)
event.start()
```
**Phương thức của object dữ liệu BidAskData:**

| Tên phương thức của object dữ liệu | Mô tả |
|---|---|
| to_dataFrame() | Trả về tất cả các thuộc tính của dữ liệu thay vì một thuộc tính riêng lẻ, được lưu trữ dưới dạng một Pandas DataFrame. |

**Thuộc tính dữ liệu của BidAskData:**

| Tên thuộc tính | Mô tả | Kiểu dữ liệu |
|---|---|---|
| ComGroupCode | Mã chỉ số | str |
| StockType | Loại chứng khoán | str |
| Ticker | Tên mã | str |
| TradingDate | Thời gian | str |
| Timestamp | Thời gian (làm tròn về giây) | str |
| Spread | Best1Ask - Best1Bid | float |
| SpreadDelta | Spread hiện tại - Spread trước | float |
| DepthImbalance | TotalBidVolume / (TotalBidVolume + TotalAskVolume) | float |
| TotalBuyTradeVolume | Tổng khối lượng giao dịch mua | int |
| TotalSellTradeVolume | Tổng khối lượng giao dịch bán | int |
| TotalBidVolume | Tổng khối lượng đặt mua | int |
| TotalAskVolume | Tổng khối lượng đặt bán | int |
| Best1Bid → Best10Bid | Bước giá mua từ 1 đến 10 | float |
| Best1Ask → Best10Ask | Bước giá bán từ 1 đến 10 | float |
| Best1BidVolume → Best10BidVolume | Khối lượng đặt mua từ mức 1 đến 10 | int |
| Best1AskVolume → Best10AskVolume | Khối lượng đặt bán từ mức 1 đến 10 | int |
| BidPriceDelta1 → BidPriceDelta10 | Chênh lệch giá mua các mức so với tick trước | float |
| AskPriceDelta1 → AskPriceDelta10 | Chênh lệch giá bán các mức so với tick trước | float |
| BidVolumeDelta1 → BidVolumeDelta10 | Chênh lệch khối lượng mua các mức so với tick trước | int |
| AskVolumeDelta1 → AskVolumeDelta10 | Chênh lệch khối lượng bán các mức so với tick trước | int |
| VWAPBid | Giá trung bình bên mua | float |
| VWAPAsk | Giá trung bình bên bán | float |
| VWAPBidSpread | Best1Bid - VWAPBid | float |
| VWAPAskSpread | VWAPAsk - Best1Ask | float |
| VWAPBidAskSpread | VWAPAsk - VWAPBid | float |
| OrderFlowImbalance | VWAPBidSpread / (VWAPBidSpread + VWAPAskSpread) | float |

** Lưu ý: Đối với các mã thuộc sàn HOSE và HNX thì chỉ có 3 mức giá mua/bán, còn đối với các mã thuộc sàn UPCOM thì có 10 mức giá mua/bán.**

**Ví dụ sử dụng:**
```python
from FiinQuantX import FiinSession, BidAskData

import pandas as pd
import time

username = 'REPLACE_WITH_YOUR_USER_NAME'
password = 'REPLACE_WITH_YOUR_PASS_WORD'

client = FiinSession(username=username, password=password).login()

tickers = ['HPG', 'VNINDEX', 'VN30F1M']

def onTickerEvent(data: BidAskData):
    print('----------------')
    print(data.to_dataFrame())

Events = client.BidAsk(tickers=tickers, callback=onTickerEvent)
Events.start()

try:
    while not Events._stop:
        time.sleep(1)
except KeyboardInterrupt:
    print("KeyboardInterrupt received, stopping...")
    Events.stop()
