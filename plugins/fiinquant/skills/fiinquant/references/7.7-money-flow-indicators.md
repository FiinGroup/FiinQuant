# 8.7.1. Multi Color Dragon Extended (MCDX)

**Tên hàm:**
- `fi.mcdx_banker(close, rsi_base_banker=50, rsi_period_banker=50, rsi_base_hot_money=30, rsi_period_hot_money=40, sensitivity_banker=1.5, sensitivity_hot=0.7, banker_ma=10)`
- `fi.mcdx_hot_money(close, rsi_base_banker=50, rsi_period_banker=50, rsi_base_hot_money=30, rsi_period_hot_money=40, sensitivity_banker=1.5, sensitivity_hot=0.7, banker_ma=10)`
- `fi.mcdx_retail(close, rsi_base_banker=50, rsi_period_banker=50, rsi_base_hot_money=30, rsi_period_hot_money=40, sensitivity_banker=1.5, sensitivity_hot=0.7, banker_ma=10)`

**Mục đích:**
Phân tích dòng tiền theo ba nhóm nhà đầu tư gồm nhà đầu tư lớn (Banker), nhà đầu cơ ngắn hạn (Hot Money) và nhà đầu tư nhỏ lẻ (Retail), từ đó hỗ trợ nhận diện xu hướng tham gia vốn trên thị trường.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| close | Cột chứa giá đóng cửa | pandas.Series |  |
| rsi_base_banker | Mức ngưỡng RSI nền của dòng tiền Banker | int | 50 |
| rsi_period_banker | Chu kỳ RSI dùng để tính dòng tiền Banker | int | 50 |
| rsi_base_hot_money | Ngưỡng RSI nền cho dòng tiền Hot Money | int | 30 |
| rsi_period_hot_money | Chu kỳ RSI cho dòng tiền Hot Money | int | 40 |
| sensitivity_banker | Hệ số nhạy điều chỉnh độ mạnh của dòng tiền Banker | float | 1.5 |
| sensitivity_hot | Hệ số nhạy cho dòng tiền Hot Money | float | 0.7 |
| banker_ma | Độ dài trung bình động để làm mượt đường Banker | int | 10 |

**Kết quả trả về (Returns):**
- `mcdx_banker()`: trả về `pandas.Series` dòng tiền Banker
- `mcdx_hot_money()`: trả về `pandas.Series` dòng tiền Hot Money
- `mcdx_retail()`: trả về `pandas.Series` dòng tiền Retail

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df["banker"] = fi.mcdx_banker(
    df["close"],
    rsi_base_banker=50,
    rsi_period_banker=50,
    rsi_base_hot_money=30,
    rsi_period_hot_money=40,
    sensitivity_banker=1.5,
    sensitivity_hot=0.7,
    banker_ma=10
)
df["hotmoney"] = fi.mcdx_hot_money(
    df["close"],
    rsi_base_banker=50,
    rsi_period_banker=50,
    rsi_base_hot_money=30,
    rsi_period_hot_money=40,
    sensitivity_banker=1.5,
    sensitivity_hot=0.7,
    banker_ma=10
)
df["retail"] = fi.mcdx_retail(
    df["close"],
    rsi_base_banker=50,
    rsi_period_banker=50,
    rsi_base_hot_money=30,
    rsi_period_hot_money=40,
    sensitivity_banker=1.5,
    sensitivity_hot=0.7,
    banker_ma=10
)
print(df)

```
# 8.7.2. Chaikin Money Flow (CMF)

**Tên hàm:** `fi.cmf(high, low, close, volume, window=20)`

**Mục đích:**
Đo lường mức độ dòng tiền chảy vào hoặc chảy ra khỏi cổ phiếu trong một khoảng thời gian nhất định bằng cách kết hợp vị trí giá đóng cửa trong biên độ giá của phiên với khối lượng giao dịch.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| high | Cột giá cao nhất | pandas.Series |  |
| low | Cột giá thấp nhất | pandas.Series |  |
| close | Giá đóng cửa | pandas.Series |  |
| volume | Khối lượng khớp lệnh | pandas.Series |  |
| window | Số phiên tính CMF (rolling window) | int | 20 |

**Kết quả trả về (Returns):**
- Trả về `pandas.Series`
- Mỗi giá trị là chỉ số CMF tương ứng tại từng thời điểm
- Giá trị dương thường phản ánh áp lực mua, giá trị âm phản ánh áp lực bán

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['cmf_20'] = fi.cmf(
    high=df['high'],
    low=df['low'],
    close=df['close'],
    volume=df['volume'],
    window=20
)
print(df)
