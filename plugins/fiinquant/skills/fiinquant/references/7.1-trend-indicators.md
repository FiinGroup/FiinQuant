# 8.1.1. EMA (Exponential Moving Average)

**Tên hàm:** `fi.ema(column, window)`

**Mục đích:**
Tính đường trung bình động hàm mũ (EMA), trong đó các giá trị gần hiện tại được gán trọng số cao hơn, giúp phản ứng nhanh hơn với biến động giá.

**Tham số:**

| Tên tham số | Ý nghĩa | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| column | Cột dữ liệu chứa các giá trị để tính toán EMA | pandas.Series |  |
| window | Số lượng điểm dữ liệu sử dụng trong phép tính EMA | int |  |

**Kết quả trả về (Returns):**
- Trả về `pandas.Series`
- Mỗi giá trị là EMA tương ứng tại từng thời điểm

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['ema_5'] = fi.ema(df['close'], window=5)
print(df)
```

# 8.1.5. MACD (Moving Average Convergence Divergence)

**Tên hàm:**
- `fi.macd(column, window_slow=26, window_fast=12)`
- `fi.macd_signal(column, window_slow=26, window_fast=12, window_sign=9)`
- `fi.macd_diff(column, window_slow=26, window_fast=12, window_sign=9)`

**Mục đích:**
Tính MACD, đường tín hiệu và histogram để đo lường động lượng, hướng và sức mạnh của xu hướng giá.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| column | Cột dữ liệu chứa các giá trị để tính toán MACD | pandas.Series |  |
| window_slow | Số lượng điểm dữ liệu cho EMA dài hạn | int | 26 |
| window_fast | Số lượng điểm dữ liệu cho EMA ngắn hạn | int | 12 |
| window_sign | Số lượng điểm dữ liệu cho EMA của đường tín hiệu | int | 9 |

**Kết quả trả về (Returns):**
- `macd()`: trả về `pandas.Series` của đường MACD
- `macd_signal()`: trả về `pandas.Series` của đường tín hiệu
- `macd_diff()`: trả về `pandas.Series` của histogram MACD

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['macd'] = fi.macd(df['close'], window_fast=12, window_slow=26)
df['macd_signal'] = fi.macd_signal(df['close'], window_fast=12, window_slow=26, window_sign=9)
df['macd_diff'] = fi.macd_diff(df['close'], window_fast=12, window_slow=26, window_sign=9)
print(df)

```
