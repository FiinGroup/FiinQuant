# 6.1. Rebalance

Mô tả cách sử dụng hàm sau khi người dùng đã đăng nhập.

**Tên hàm:** `client.Rebalance().get(Budget=..., Ticker=...)`

**Mục đích:**
Bằng cách nhập giá trị có thể mua và mã chỉ số, thuật toán sẽ tính ra số lượng cổ phiếu cần mua để có tỷ lệ gần nhất với tỷ lệ cổ phiếu trong rổ chỉ số tương ứng. Hàm này phù hợp cho các chiến lược giao dịch arbitrage và passive investment.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Mặc định | Bắt buộc |
|---|---|---|---|---|
| Budget | Ngân sách đầu tư tính bằng VND dùng để phân bổ vào danh mục mô phỏng theo rổ chỉ số | int hoặc float |  | Có |
| Ticker | Mã chỉ số cần rebalance | str |  | Có |

**Danh sách các Index dùng cho hàm:**
- `VN30`
- `VN100`
- `VNX50`
- `VNMidcap`
- `VNFINSELECT`
- `VNDIAMOND`
- `VNFINLEAD`

**Phương pháp luận:**
- Hàm `Rebalance` sử dụng dữ liệu công bố của Sở giao dịch chứng khoán về:
  - Tỷ lệ FreeFloat
  - Giới hạn tỷ trọng
  - Giá đóng cửa realtime của hệ thống FiinGroup

- Tái cơ cấu được thực hiện theo nguyên tắc tỷ lệ về **khối lượng không thay đổi trong một danh mục**.
- Để bám theo một chỉ số, nhà quản trị danh mục cần xây dựng bộ danh mục có tỷ lệ về **số lượng giữa các cổ phiếu** giống với tỷ lệ trong rổ chỉ số cần theo dõi.
- Cách tiếp cận này giúp đảm bảo độ lệch so với chỉ số gốc (Tracking Error) ở mức thấp nhất.

**Kết quả trả về (Returns):**
- Dữ liệu dạng bảng (`DataFrame`)
- Kết quả trả ra là danh sách cổ phiếu trong rổ chỉ số cùng với số lượng cổ phiếu cần mua tương ứng theo ngân sách đầu vào
- Trường quan trọng nhất là **`Shares to Buy`**, thể hiện số lượng cần mua của từng mã

**Thuộc tính dữ liệu:**

| Tên thuộc tính | Mô tả | Kiểu dữ liệu |
|---|---|---|
| Ticker | Mã cổ phiếu | str |
| Timestamp | Thời điểm dữ liệu | str |
| Close | Giá đóng cửa / giá realtime dùng để tính toán | float |
| FreeFloat | Số lượng free-float | float |
| OutstandingShare | Số lượng cổ phiếu lưu hành | float |
| FreeFloatRate | Tỷ lệ free-float | float |
| MarketCapLimit | Giới hạn tỷ trọng vốn hóa | float |
| TotalCapitalization | Vốn hóa tính toán theo free-float và giới hạn | float |
| Stock Ratio | Tỷ trọng cổ phiếu trong danh mục mô phỏng | float |
| Shares to Buy | Số lượng cổ phiếu cần mua | float |
| Stock Value to Buy | Giá trị tiền cần phân bổ để mua mã cổ phiếu đó | float |

**Ví dụ sử dụng:**

```python
from FiinQuantX import FiinSession

username = 'REPLACE_WITH_YOUR_USER_NAME'
password = 'REPLACE_WITH_YOUR_PASS_WORD'

client = FiinSession(username=username, password=password).login()

df = client.Rebalance().get(Budget=10000000000, Ticker='VN30')
print(df)
```
