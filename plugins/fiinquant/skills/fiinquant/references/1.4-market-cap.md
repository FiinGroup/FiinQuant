# Lấy dữ liệu vốn hóa

**Tên hàm:** `client.PriceStatistics().get_overview(...)`

**Mục đích:**
Lấy các dữ liệu có thông tin vĩ mô của riêng từng mã cổ phiếu (vốn hóa, khối lượng/giá trị giao dịch, biến động giá...) của từng mã cổ phiếu.

**Tham số (Parameters):**

| Tham số | Mô tả | Kiểu dữ liệu | Mặc định | Bắt buộc |
|---------|-------|---------|---------|---------|
| tickers | Danh sách mã được viết in hoa. | str hoặc list | None | Có |
| time_filter | Tần suất lấy dữ liệu, có thể lấy dữ liệu theo Ngày, Tuần, Tháng, Quý, Năm. Truyền vào 1 trong 5 giá trị "Daily", "Weekly", "Monthly", "Quarterly", "Yearly". | str | None | Có |
| from_date | Mốc thời gian lấy dữ liệu xa nhất (date string có format "%Y-%m-%d"). | str | None | Có |
| to_date | Mốc thời gian lấy dữ liệu gần nhất (date string có format "%Y-%m-%d"). | str | datetime.now().strftime("%Y-%m-%d") | Không |

**Kết quả trả về (Returns):**
- Kiểu dữ liệu là `pd.Dataframe` có các thuộc tính sau:

| Tham số | Mô tả | Kiểu dữ liệu |
|---------|-------|---------|
| ticker | Tên mã. | str |
| timestamp | Thời gian giao dịch (date string có format "%Y-%m-%d 00:00") - chỉ xuất hiện khi time_filter="Daily" | str |
| week | Tuần trong năm - chỉ xuất hiện khi time_filter="Weekly" | int |
| month | Tháng trong năm - chỉ xuất hiện khi time_filter="Monthly" | int |
| quarter | Quý trong năm - chỉ xuất hiện khi time_filter="Quarterly" | int |
| year | Năm - chỉ xuất hiện khi time_filter nhận 1 trong 4 giá trị "Weekly", "Monthly", "Quarterly", "Yearly" | int |
| percentPriceChange | % thay đổi giá đóng cửa so với giá tham chiếu | float |
| totalMatchVolume | Tổng khối lượng khớp lệnh | float |
| totalMatchValue | Tổng giá trị khớp lệnh | int |
| totalDealVolume | Tổng khối lượng thỏa thuận | float |
| totalDealValue | Tổng giá trị thỏa thuận | int |
| marketCap | Tổng giá trị vốn hóa | int |

**Ví dụ:**
```python
from FiinQuantX import FiinSession

username = 'REPLACE_WITH_YOUR_USER_NAME'
password = 'REPLACE_WITH_YOUR_PASS_WORD'

client = FiinSession(username=username, password=password).login()

tickers = ["HPG", "MWG"]

data = client.PriceStatistics().get_overview(
    tickers=tickers,
    time_filter="Daily",
    from_date="2025-08-20",
    to_date="2025-08-27"
)
print(data)
