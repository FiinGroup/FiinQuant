# 7.8. Danh sách lệnh

**Tên hàm:**
- Cơ sở: `orderbook.get_list_orders(...)`
- Phái sinh: `orderbook_der.get_list_orders(...)`

**Mục đích:**
Lấy danh sách các lệnh đã tạo của tài khoản giao dịch, hỗ trợ lọc theo khoảng thời gian, mã chứng khoán hoặc mã hợp đồng và trạng thái lệnh.

**Tham số:**

| Tên tham số | Kiểu dữ liệu | Bắt buộc | Mô tả |
|---|---|---|---|
| from_date | str | Không | Thời gian lọc ngày nhỏ nhất. Nếu không truyền, trả về tất cả các lệnh từ ngày cũ nhất trong danh sách |
| to_date | str | Không | Thời gian lọc ngày lớn nhất. Nếu không truyền, trả về tất cả các lệnh đến ngày mới nhất trong danh sách |
| ticker | str | Không | Lọc theo mã. Nếu không truyền, trả về tất cả các mã có trong danh sách lệnh |
| status | str | Không | Lọc theo trạng thái. Nếu không truyền, trả về tất cả trạng thái hiện có trong danh sách |

**Kết quả trả về (Returns):**
- Trả về danh sách object lệnh
- Mỗi phần tử trong danh sách biểu diễn một lệnh giao dịch
- Có thể gọi `.summary()` trên từng object để hiển thị nhanh thông tin

**Thuộc tính dữ liệu:**

| Tên thuộc tính | Kiểu dữ liệu | Mô tả |
|---|---|---|
| id | str | Mã định danh của lệnh |
| ticker | str | Mã chứng khoán hoặc mã phái sinh |
| side | str | Hướng giao dịch: mua hoặc bán |
| order_type | str | Loại lệnh, ví dụ: `LO`, `MTL`, `ATC`, ... |
| price | float | Giá đặt lệnh |
| quantity | int | Khối lượng đặt lệnh |
| filled | int | Tổng khối lượng đã khớp |
| avg_price | float | Giá khớp trung bình của lệnh |
| modified | datetime | Thời điểm sửa đổi hoặc cập nhật lệnh |
| created | datetime | Thời điểm tạo lệnh |
| status | str | Trạng thái lệnh |
| error | str hoặc None | Thông tin lỗi (nếu có) |
| status_code | int | Mã code, nếu thành công là 200 |

**Ví dụ cho cơ sở:**

```python
username_dnse = 'REPLACE_WITH_YOUR_DNSE_USERNAME'
password_dnse = 'REPLACE_WITH_YOUR_DNSE_PASSWORD'

client_order = client.FiinQuantConnector(
    broker='DNSE',
    username=username_dnse,
    password=password_dnse,
    smart_otp=True
).login()

orderbook = client_order.get_orderbook(
    account_id='0001009212',
    loan_id='13720'
)

order_list = orderbook.get_list_orders(
    from_date='2025-10-09',
    to_date='2025-10-16'
)

for i in order_list:
    print(i.summary())
```
**Output mẫu cho cơ sở:**
```python
{
    "id": 191,
    "ticker": "HPG",
    "side": "Buy",
    "order_type": "LO",
    "price": 26900.0,
    "quantity": 1000,
    "filled": 0,
    "avg_price": 0.0,
    "modified": "29/10/2025 11:23:54",
    "created": "29/10/2025 11:23:54",
    "status": "New",
    "error": null,
    "status_code": 200
}
```
**Ví dụ cho phái sinh:**
```python
username_dnse = 'REPLACE_WITH_YOUR_DNSE_USERNAME'
password_dnse = 'REPLACE_WITH_YOUR_DNSE_PASSWORD'

client_order = client.FiinQuantConnector(
    broker='DNSE',
    username=username_dnse,
    password=password_dnse,
    smart_otp=True
).login()

orderbook_der = client_order.get_derivative_orderbook(
    account_id='0001009212',
    loan_id='2278'
)

derivative_order_list = orderbook_der.get_list_orders(
    from_date='2025-10-09',
    to_date='2025-10-16'
)

for i in derivative_order_list:
    print(i.summary())
```
