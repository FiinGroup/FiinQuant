# 8.3.1. Bollinger Bands

**Tên hàm:**
- `fi.bollinger_mband(column, window=20)`
- `fi.bollinger_hband(column, window=20, window_dev=2)`
- `fi.bollinger_lband(column, window=20, window_dev=2)`
- `fi.bollinger_wband(column, window=20, window_dev=2)`
- `fi.bollinger_squeeze(column, window=20, window_dev=2, squeeze_mode="dynamic", dynamic_thresold=0.9, percentile_thresold=10)`

**Mục đích:**
Đo lường mức độ biến động của giá và xác định các dải trên, dải dưới, độ rộng dải cũng như trạng thái nén giá của Bollinger Bands.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| column | Cột dữ liệu chứa các giá trị để tính toán Bollinger Bands | pandas.Series |  |
| window | Số lượng điểm dữ liệu sử dụng trong phép tính Bollinger Bands | int | 20 |
| window_dev | Số lượng độ lệch chuẩn sử dụng để tính toán khoảng cách các dải | int | 2 |
| squeeze_mode | Kiểu nén giá: `'dynamic'` hoặc `'percentile'` | str | `'dynamic'` |
| dynamic_thresold | Hệ số kiểu nén giá linh động | float | 0.9 |
| percentile_thresold | Hệ số kiểu nén giá percentile | int | 10 |

**Kết quả trả về (Returns):**
- `bollinger_mband()`: trả về `pandas.Series` của dải giữa
- `bollinger_hband()`: trả về `pandas.Series` của dải trên
- `bollinger_lband()`: trả về `pandas.Series` của dải dưới
- `bollinger_wband()`: trả về `pandas.Series` độ rộng dải
- `bollinger_squeeze()`: trả về `pandas.Series` trạng thái squeeze

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['bollinger_mband'] = fi.bollinger_mband(df['close'], window=20)
df['bollinger_hband'] = fi.bollinger_hband(df['close'], window=20, window_dev=2)
df['bollinger_lband'] = fi.bollinger_lband(df['close'], window=20, window_dev=2)
df['bollinger_wband'] = fi.bollinger_wband(df['close'], window=20, window_dev=2)
df['bollinger_squeeze'] = fi.bollinger_squeeze(df['close'], window=20, window_dev=2, squeeze_mode='dynamic', dynamic_thresold=0.9)
print(df)

```
# 8.3.2. Supertrend

**Tên hàm:**
- `fi.supertrend(high, low, close, window=14, multiplier=3.0)`
- `fi.supertrend_hband(high, low, close, window=14, multiplier=3.0)`
- `fi.supertrend_lband(high, low, close, window=14, multiplier=3.0)`

**Mục đích:**
Xác định xu hướng chính của thị trường và các điểm đảo chiều tiềm năng bằng cách kết hợp ATR và biến động giá.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| high | Cột dữ liệu chứa các giá trị giá cao nhất để tính toán Supertrend | pandas.Series |  |
| low | Cột dữ liệu chứa các giá trị giá thấp nhất để tính toán Supertrend | pandas.Series |  |
| close | Cột dữ liệu chứa các giá trị giá đóng cửa để tính toán Supertrend | pandas.Series |  |
| window | Số lượng điểm dữ liệu sử dụng trong phép tính Supertrend | int | 14 |
| multiplier | Hệ số nhân dùng để điều chỉnh độ rộng của chỉ báo Supertrend | float | 3.0 |

**Kết quả trả về (Returns):**
- `supertrend()`: trả về `pandas.Series` chỉ báo Supertrend
- `supertrend_hband()`: trả về `pandas.Series` dải trên
- `supertrend_lband()`: trả về `pandas.Series` dải dưới

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['supertrend'] = fi.supertrend(df['high'], df['low'], df['close'], window=14)
df['supertrend_hband'] = fi.supertrend_hband(df['high'], df['low'], df['close'], window=14)
df['supertrend_lband'] = fi.supertrend_lband(df['high'], df['low'], df['close'], window=14)
print(df)

```
# 8.3.3. ATR (Average True Range)

**Tên hàm:** `fi.atr(high, low, close, window=14)`

**Mục đích:**
Đo lường mức độ biến động thực tế của giá trong một khoảng thời gian nhất định để hỗ trợ đánh giá rủi ro và thiết lập mức dừng lỗ hoặc chốt lời.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| high | Cột dữ liệu chứa các giá trị giá cao nhất để tính toán ATR | pandas.Series |  |
| low | Cột dữ liệu chứa các giá trị giá thấp nhất để tính toán ATR | pandas.Series |  |
| close | Cột dữ liệu chứa các giá trị giá đóng cửa để tính toán ATR | pandas.Series |  |
| window | Số lượng điểm dữ liệu sử dụng trong phép tính ATR | int | 14 |

**Kết quả trả về (Returns):**
- Trả về `pandas.Series`
- Mỗi giá trị là ATR tại từng thời điểm

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['atr'] = fi.atr(df['high'], df['low'], df['close'], window=14)
print(df)

```
# 8.3.4. Realized Volatility

**Tên hàm:** `fi.realized_volatility(close, window=14, trading_days=252)`

**Mục đích:**
Đo lường mức độ biến động thực tế của giá dựa trên lợi suất logarit trong một khoảng thời gian và annualize theo năm.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| close | Cột dữ liệu chứa các giá trị giá đóng cửa để tính realized volatility | pandas.Series |  |
| window | Số phiên được dùng để tính rolling volatility | int | 14 |
| trading_days | Số ngày giao dịch trong năm dùng để annualize | int | 252 |

**Kết quả trả về (Returns):**
- Trả về `pandas.Series`
- Mỗi giá trị là realized volatility đã quy đổi theo năm

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df['rv'] = fi.realized_volatility(df["close"], window=14, trading_days=252)
print(df)

```
# 8.3.5. Keltner Channels

**Tên hàm:**
- `fi.kc_midline(close, high, low, ema_window=20, atr_window=10, atr_mult=2)`
- `fi.kc_upper(close, high, low, ema_window=20, atr_window=10, atr_mult=2, band_style='avg_true_range')`
- `fi.kc_lower(close, high, low, ema_window=20, atr_window=10, atr_mult=2, band_style='avg_true_range')`

**Mục đích:**
Tính các kênh Keltner để xác định vùng biến động giá, đánh giá xu hướng, phát hiện phá kênh và xác định vùng quá mua/quá bán.

**Tham số:**

| Tên tham số | Mô tả | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| close | Cột dữ liệu chứa các giá trị giá đóng cửa để tính toán Keltner Channels | pandas.Series |  |
| high | Cột dữ liệu chứa các giá trị giá cao nhất | pandas.Series |  |
| low | Cột dữ liệu chứa các giá trị giá thấp nhất | pandas.Series |  |
| ema_window | Số phiên dùng để tính đường EMA | int | 20 |
| atr_window | Số phiên dùng để tính ATR | int | 10 |
| atr_mult | Hệ số nhân với ATR để tính kênh trên và kênh dưới | float | 2 |
| band_style | Kiểu dải: `'avg_true_range'`, `'true_range'`, `'range'` | str | `'avg_true_range'` |

**Kết quả trả về (Returns):**
- `kc_midline()`: trả về `pandas.Series` đường giữa
- `kc_upper()`: trả về `pandas.Series` dải trên
- `kc_lower()`: trả về `pandas.Series` dải dưới

**Ví dụ sử dụng:**

```python
fi = client.FiinIndicator()
df["kc_midline"] = fi.kc_midline(df["close"], df["high"], df["low"], ema_window=20, atr_window=10, atr_mult=2)
df["kc_upper"] = fi.kc_upper(df["close"], df["high"], df["low"], ema_window=20, atr_window=10, atr_mult=2, band_style='avg_true_range')
df["kc_lower"] = fi.kc_lower(df["close"], df["high"], df["low"], band_style='avg_true_range')
print(df)
