# Lấy dữ liệu giá trần, giá sàn

**Tên hàm:** `client.PriceStatistics().get_ceilingfloor(...)`

**Mục đích:**
Lấy dữ liệu giá trần và giá sàn của một hoặc nhiều mã chứng khoán trong khoảng thời gian xác định.

**Tham số (Parameters):**

| Tham số | Mô tả | Kiểu dữ liệu | Mặc định | Bắt buộc |
|---------|-------|---------|---------|---------|
| tickers | Danh sách mã được viết in hoa. | str hoặc list | None | Có |
| from_date | Mốc thời gian lấy dữ liệu xa nhất (date string có format "%Y-%m-%d"). | str | None | Có |
| to_date | Mốc thời gian lấy dữ liệu gần nhất (date string có format "%Y-%m-%d"). | str | datetime.now().strftime("%Y-%m-%d") | Không |

**Kết quả trả về (Returns):**
- Kiểu dữ liệu là `pd.Dataframe` có các thuộc tính sau:
| Tham số | Mô tả | Kiểu dữ liệu |
|---------|-------|---------|
| ticker | Tên mã. | str |
| timestamp | Thời gian giao dịch (date string có format "%Y-%m-%d 00:00") | str |
| floorValue | Giá sàn | int |
| ceilingValue | Giá trần | int |

**Ví dụ:**
```python
from FiinQuantX import FiinSession

username = 'REPLACE_WITH_YOUR_USER_NAME'
password = 'REPLACE_WITH_YOUR_PASS_WORD'

client = FiinSession(username=username, password=password).login()

tickers = ["HPG", "MWG"]

data = client.PriceStatistics().get_ceilingfloor(
    tickers=tickers,
    from_date="2025-10-01",
    to_date="2025-10-08"
)

print(data)
