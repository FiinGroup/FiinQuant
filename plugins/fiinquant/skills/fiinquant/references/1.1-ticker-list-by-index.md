# Danh sách mã theo index

**Tên hàm:** client.TickerList(ticker='INDEX_NAME')

**Mục đích:**
Bằng cách nhập vào tên INDEX (Ví dụ: VN30, VN100, VNINDEX, HNXINDEX, v.v.) hàm sẽ trả ra 1 danh sách các mã thuộc INDEX đó, điều này sẽ thuận tiện cho người dùng muốn gọi dữ liệu lịch sử của các mã thuộc cùng 1 rổ INDEX.

**Tham số (Parameters):**
| Tham số | Mô tả | Kiểu dữ liệu | Mặc định | Bắt buộc |
|---------|-------|---------|---------|---------|
| ticker | tên INDEX (Ví dụ: VN30, VN100, VNINDEX, HNXINDEX, v.v.) | str | None | Không |
| tickers | Danh sách tên INDEX, được dùng nếu muốn lấy danh sách các mã thuộc 2 INDEX cùng 1 lúc | List of str | None | Không |
**Lưu ý:**
Chỉ được dùng 1 trong 2 tham số, không được dùng cùng lúc, 1 là gọi danh sách các mã cho 1 INDEX thì dùng ticker, nếu muốn gọi cho từ 2 INDEX trở lên thì dùng tickers.

Danh sách các index để truyền vào hàm TickerList sẽ được mô tả ở mục `references/code_lists.md`

**Kết quả trả về (Returns):**
- Kiểu dữ liệu: `một instance của class TickerList`. Class `TickerList` có implement các method:
    + `__iter__(self)`: return str(self.data)
    + `__len__(self)`: return len(self.data)
    + `__getitem__(self, index)`: return self.data[index]
    + `__contains__(self, item)`: return item.upper() in self.data
    
Trong đó `self.data` có data type là List[str]. Điều này khiến object behave giống như một list.

Cách chuyển sang list thông thường:
```python
list(client.TickerList(ticker='VN30'))
```

**Ví dụ:**
```python
from FiinQuantX import FiinSession

username = 'REPLACE_WITH_YOUR_USERNAME'
password = 'REPLACE_WITH_YOUR_PASSWORD'

client = FiinSession(username=username, password=password).login()

# Lấy danh sách mã thuộc VN30
tickers = client.TickerList(ticker='VN30')
print(type(tickers))
lst_tickers = list(tickers)
print(lst_tickers)
# Output: <class 'FiinQuantX.helper.TickerList.TickerList'>
```
