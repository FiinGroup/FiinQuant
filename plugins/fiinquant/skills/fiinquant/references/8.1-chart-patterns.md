# 9.1.1. Mô hình vai đầu vai (Head and Shoulders Pattern)

**Tên hàm:** `pa.detect_head_shoulder(df, window=10, tolerance=0.01)`

**Mục đích:**
Nhận diện mô hình vai đầu vai, một mô hình đảo chiều xu hướng phổ biến thường xuất hiện ở cuối xu hướng tăng và báo hiệu khả năng chuyển sang xu hướng giảm.

**Tham số:**

| Tên tham số | Ý nghĩa | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| df | DataFrame chứa dữ liệu để xác định mô hình, bắt buộc có các cột `high`, `low`, `close` | pandas.DataFrame |  |
| window | Độ rộng nến để xác định các đỉnh hoặc đáy cục bộ của mô hình | int | 10 |
| tolerance | Tỷ lệ % tối thiểu mà điểm đầu phải cao hơn hai vai | float | 0.01 |

**Kết quả trả về (Returns):**
- Trả về `pandas.Series`
- Series sẽ đánh dấu các vị trí thuộc mô hình bằng các nhãn:
  - `Entry`
  - `Left_Shoulder`
  - `Left_Neck`
  - `Head`
  - `Right_Neck`
  - `Right_Shoulder`

**Ví dụ sử dụng:**

```python
username = "REPLACE_WITH_YOUR_USERNAME"
password = "REPLACE_WITH_YOUR_PASSWORD"

client = FiinSession(username=username, password=password).login()

df = client.Fetch_Trading_Data(
    realtime=False,
    tickers=["HPG"],
    fields=["high","low","close"],
    adjusted=True,
    by="1d",
    from_date="2024-01-01",
    to_date="2024-12-31"
).get_data()

pa = client.Pattern()
df["head_shoulder"] = pa.detect_head_shoulder(df)
print(df)
```

# 9.1.2. Mô hình vai đầu vai ngược (Inverse Head and Shoulders Pattern)

**Tên hàm:** `pa.detect_inverse_head_shoulder(df, window=10, tolerance=0.01)`

**Mục đích:**
Nhận diện mô hình vai đầu vai ngược, một mô hình đảo chiều dùng để dự báo sự chuyển đổi từ xu hướng giảm sang xu hướng tăng.

**Tham số:**

| Tên tham số | Ý nghĩa | Kiểu dữ liệu | Giá trị mặc định |
|---|---|---|---|
| df | DataFrame chứa dữ liệu để xác định mô hình, bắt buộc có các cột `high`, `low`, `close` | pandas.DataFrame |  |
| window | Độ rộng nến để xác định các đỉnh hoặc đáy cục bộ của mô hình | int | 10 |
| tolerance | Tỷ lệ % tối thiểu mà điểm đầu phải thấp hơn hai vai theo điều kiện mô hình | float | 0.01 |

**Kết quả trả về (Returns):**
- Trả về `pandas.Series`
- Series sẽ đánh dấu các vị trí thuộc mô hình vai đầu vai ngược bằng các nhãn tương ứng của vai trái, đầu, vai phải và vùng neckline

**Ví dụ sử dụng:**

```python
pa = client.Pattern()
df["inverse_head_shoulder"] = pa.detect_inverse_head_shoulder(df, window=10, tolerance=0.01)
print(df)
